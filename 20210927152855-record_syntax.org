#+title: record syntax
#+filetags: :haskell:

You can define a type using record syntax. This allows you to provide
field names to each type in the constructor.

#+BEGIN_SRC haskell
data Employee =
  Employee
    { employeeName :: String
    , employeeAge :: Int
    }
#+END_SRC
