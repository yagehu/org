#+title: Haskell typeclass

* Typeclass

Typeclasses define a set of functions that can have different
implementations depending on the type of data they are given.

** Example

#+BEGIN_SRC haskell
  class BasicEq a where
    isEqual :: a -> a -> Bool
#+END_SRC
